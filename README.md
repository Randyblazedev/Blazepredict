# BlazePredict

Football match predictions powered by data from football-data.org.

## Setup

1. Get a free API key from [football-data.org](https://www.football-data.org/)
2. Open `blazepredict.js` and replace the API token

## ⚠️ Note

The API key is client-side (frontend JS) since this is a static site. In production, you'd want to proxy requests through a backend to protect the key.
